# AMO Listing Description — Claude Status Monitor v2.0

## Kurzbeschreibung (max. 250 Zeichen)

Zeigt den Live-Status der Anthropic-Dienste direkt auf claude.ai – als Widget und als detailliertes Toolbar-Popup. Mit 7-Tage-Verlauf, Incidents und Wartungsfenstern. Deutsch & Englisch.

---

## Vollständige Beschreibung

**Claude Status Monitor** zeigt dir in Echtzeit, ob die Claude-Dienste von Anthropic verfügbar sind – ohne den Browser zu verlassen.

### Widget auf claude.ai

- **Live-Statusanzeige** — erscheint unten rechts auf claude.ai
- **Farbkodierter Indikator** — grün (alles ok), orange (eingeschränkt/Teilausfall), rot (Totalausfall), grau (unbekannt/Wartung)
- **Aufklappbare Komponentenliste** — Status jedes einzelnen Anthropic-Dienstes auf einen Blick
- **Auto-Refresh** — aktualisiert sich automatisch alle 60 Sekunden

### Toolbar-Popup

- **Detaillierte Statusübersicht** — Klick auf das Erweiterungssymbol öffnet das vollständige Dashboard
- **Alle Dienstkomponenten** mit aktuellem Status
- **7-Tage-Verlaufschart** — farbkodierte Balken zeigen den Incident-Verlauf pro Tag
- **Aktive Incidents** — mit Schweregrad, Status und letzter Aktualisierung
- **Wartungsfenster** — geplante und laufende Wartungsarbeiten
- **Incident-Chronik** — abgeschlossene Incidents der letzten 7 Tage mit Dauer

### Allgemein

- **Zweisprachig** — Deutsch 🇩🇪 und Englisch 🇺🇸, umschaltbar in Widget und Popup
- **Einstellungen bleiben erhalten** — Sprachauswahl und Widget-Zustand werden seitenübergreifend gespeichert
- **Kein Tracking, keine externen Server** — die einzige Verbindung geht an `status.anthropic.com`
- **Keine Datensammlung** — alle Einstellungen bleiben lokal im Browser

## English Version

**Claude Status Monitor** shows you in real time whether Anthropic's Claude services are available — without leaving your browser.

### Widget on claude.ai

- **Live status widget** — appears in the bottom-right corner of claude.ai
- **Color-coded indicator** — green (operational), orange (degraded/partial outage), red (major outage), gray (unknown/maintenance)
- **Expandable component list** — shows the status of each individual Anthropic service
- **Auto-refresh** — polls the official Anthropic status API every 60 seconds

### Toolbar Popup

- **Detailed status overview** — click the extension icon in the Firefox toolbar
- **All service components** with live status
- **7-day uptime chart** — color-coded bars showing incident history per day
- **Active incidents** — with impact level, status, and latest update
- **Scheduled maintenance** — upcoming and in-progress maintenance windows
- **Incident history** — resolved incidents from the last 7 days with duration

### General

- **Bilingual** — German 🇩🇪 and English 🇺🇸, switchable in both widget and popup
- **Persistent settings** — language choice and widget state are saved across page reloads
- **No tracking, no external servers** — only communicates with `status.anthropic.com`
- **No data collection** — all settings stay local in the browser
